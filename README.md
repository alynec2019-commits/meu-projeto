# Alyne & Anthony

Um mini site desenvolvido em HTML e CSS para contar um pouco da minha história com meu filho, Anthony. O projeto foi criado com o objetivo de praticar desenvolvimento web enquanto compartilho alguns dos momentos mais importantes da nossa jornada juntos.

---

## 📖 Sobre o projeto

O site apresenta uma narrativa simples e emocionante sobre a maternidade, desde a descoberta da gravidez até os desafios e alegrias vividos ao lado do Anthony.

Além de ser um projeto pessoal, ele também demonstra a aplicação de conceitos fundamentais de HTML semântico e CSS responsivo.

---

## Funcionalidades

- Página inicial apresentando a história.
- Página "Sobre" com acontecimentos marcantes.
- Galeria de imagens.
- Vídeos incorporados.
- Página de contato com formulário.
- Formulário para sugestões de novos temas.

---

## Tecnologias utilizadas

- HTML
- CSS
- Google Fonts

## Objetivo

Este projeto foi desenvolvido para colocar em prática conhecimentos de:

- Estruturação semântica com HTML;
- Estilização utilizando CSS;
- Boas práticas de desenvolvimento web.

Ao mesmo tempo, ele representa uma forma de registrar momentos especiais da minha vida e da minha relação com meu filho.

---

##  Páginas

###  Home

Apresenta uma introdução da nossa história, desde o início da gravidez até o nascimento do Anthony.

###  Sobre

Conta acontecimentos importantes da nossa jornada, incluindo desafios, brincadeiras, aprendizados e momentos inesquecíveis, utilizando textos, imagens e vídeos.

###  Contato

Permite que o visitante envie uma mensagem ou sugira novos temas para futuras histórias.

---


##  Mensagem

Mais do que um exercício de programação, este projeto representa uma lembrança de uma das fases mais importantes da minha vida.

Cada página foi desenvolvida com carinho para registrar um pequeno pedaço da nossa história, mostrando que, mesmo nos dias difíceis, o amor sempre foi o nosso maior aprendizado.
